# canaria-5gamers
Data processing &amp; analysis of the Canaria 5 Gamers dataset


### Getting Started
```sh
$ git clone https://github.com/sheriefkhorshid/canaria-5gamers.git
$ cd canaria-5gamers
$ conda env create -f environment.yml
$ jupyter notebook
```
